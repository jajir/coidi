1.  Overview

OpenID Attribute Exchange ([OpenID.attribute‑exchange‑1.0] (Hardt, D. and B. Ferg, “OpenID Attribute Exchange,” August 2006.)) is a service for OpenID ([OpenID.authentication‑2.0] (Recordon, D., Hoyt, J., and B. Fitzpatrick, “OpenID Authentication 2.0 - Draft 08,” August 2006.)) that enables transport of personal identity information. This document provides a list of standard attribute property names for use with the attribute exchange service.

Other attribute types may be defined elsewhere; this list serves as a basic standard reference.
